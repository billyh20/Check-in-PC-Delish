📋 วิธีเปิดอนุญาต GPS
Chrome (Android):
แตะไอคอน 🔒 ใน URL bar → Permissions → Location → Allow

Safari (iPhone/iPad):
Settings → Safari → Location → Allow
หรือ แตะ AA ใน URL bar → Website Settings → Location

Chrome (Desktop):
คลิกไอคอน 🔒 ใน URL bar → Site settings → Location → Allow

หลังจากเปิดแล้ว กด "🔄 ลองใหม่" บนหน้าเว็บ

⚙ ตั้งค่า Google Sheets

1. เปิด Google Sheet → Extensions → Apps Script
2. วางโค้ดจากไฟล์ apps_script.gs
3. Deploy → New deployment → Web app
4. Execute as: Me | Access: Anyone
5. Copy URL มาวางที่นี่
